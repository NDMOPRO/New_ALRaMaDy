- Added native OOXML chart embedding to exported workbooks:
  - chart part under `xl/charts/chart1.xml`
  - drawing parts under `xl/drawings/drawing1.xml` and `xl/drawings/_rels/drawing1.xml.rels`
  - worksheet relationship wiring plus anchor range injection
- Added native OOXML pivot hardening:
  - pivot table definition under `xl/pivotTables/pivotTable1.xml`
  - pivot cache definition/records under `xl/pivotCache/pivotCacheDefinition1.xml` and `xl/pivotCache/pivotCacheRecords1.xml`
  - workbook pivot cache relationship wiring and worksheet pivot table part wiring
- Added workbook-defined lambda persistence:
  - hidden `__lambda_registry` worksheet in the sample workbook
  - extraction into `lambda-registry.json`
  - workbook defined-name injection into `xl/workbook.xml`
- Added richer dynamic-array execution:
  - `FILTER`
  - `UNIQUE`
  - `SORT`
  - `XMATCH`
  - `BYROW`
  - `BYCOL`
  - `MAP`
  - `SCAN`
  - `REDUCE`
- Added broader transformations:
  - `split_sheet`
  - `merge_workbooks`
- Added `.xls` ingest through `xlsx` BIFF8 read and conversion back into ExcelJS workbook state
- Added stronger backend publication persistence under `.runtime/excel-engine-backend/publications/<publication-id>/`
