# NDMO Brand Identity Analysis

## Primary Fonts
- **Arabic**: DIN NEXT™ ARABIC (Regular, Light, Medium) — mapped to `din-next-lt-w23-regular.ttf`, `din-next-lt-w23-light.ttf`, `din-next-lt-w23-medium.ttf`
- **English**: Objectivity (Regular, Bold, Medium) — from SDAIA fonts
- **Fallback Arabic**: HelveticaNeueLT Arabic 55 Roman — `HelveticaNeueLTArabic-Roman.ttf`

## Primary Color Palette (from PPTX analysis)
- **Deep Navy**: #131938, #1D2654, #263269, #273370, #2A3878 (backgrounds, headers)
- **Primary Blue**: #46448C, #3E518E, #507DBC, #687CBD (accents, buttons)
- **Green Accent**: #2FAE80, #29BA74, #15A776, #0CAB8F (success, highlights)
- **Gold/Amber**: #FCD12A, #FCBD4C, #C29A21 (warnings, emphasis)
- **Light BG**: #F6F9FB, #EEF1F6, #E5EEFA, #E0E3F4 (content backgrounds)
- **Text**: #191919, #2E2E2E, #575757 (body text)
- **White**: #FFFFFF (slide backgrounds, text on dark)

## Key Design Patterns
- 121 slides total — professional government report style
- RTL layout throughout
- Clean, structured layouts with clear hierarchy
- Section dividers with deep navy backgrounds
- Content slides with light backgrounds
- Charts and data visualizations with the blue/green palette
