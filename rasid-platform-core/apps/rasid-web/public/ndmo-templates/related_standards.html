STD-MD-01
معيار البيانات الوصفية (Metadata Standard)
يحدد الحقول الإلزامية والاختيارية لوصف مجموعات البيانات لضمان قابليتها للاكتشاف والفهم.
المتطلبات الرئيسية:
استخدام قاموس دبلن كور (Dublin Core) · تحديد المالك وتاريخ التحديث · تصنيف البيانات حسب القائمة الوطنية
تاريخ الإصدار: 2023/05 معتمد
STD-INT-02
معيار قابلية التشغيل البيني (Interoperability)
يوحد بروتوكولات وصيغ تبادل البيانات بين الجهات الحكومية لضمان التكامل السلس.
المتطلبات الرئيسية:
اعتماد صيغ JSON / XML للتبادل · استخدام واجهات API موثقة (RESTful) · الالتزام بمعايير الترميز الموحدة (UTF-8)
تاريخ الإصدار: 2023/08 معتمد
STD-GEO-03
معيار البيانات الجيومكانية (Geospatial Data)
يضبط مواصفات البيانات المكانية والخرائط لضمان دقة الموقع والتوافق مع المرجع الوطني.
المتطلبات الرئيسية:
المرجع الجيوديسي الوطني (KSA-GRF) · صيغ الملفات المكانية (Shapefile, GeoJSON) · دقة الإحداثيات والبيانات الوصفية المكانية
تاريخ الإصدار: 2024/01 معتمد
STD-REF-04
معيار البيانات المرجعية (Reference Data)
يوحد قوائم الرموز والتصنيفات الوطنية (مثل رموز المناطق، الجنسيات، الأنشطة الاقتصادية).
المتطلبات الرئيسية:
استخدام القوائم الوطنية المعتمدة حصراً · آلية تحديث ومزامنة القوائم المرجعية · منع إنشاء قوائم محلية بديلة
تاريخ الإصدار: 2024/03 مسودة نهائية
STD-OD-05
مواصفات نشر البيانات المفتوحة
يحدد الشروط الفنية لنشر البيانات للعموم لضمان سهولة الاستخدام وإعادة الاستخدام.
المتطلبات الرئيسية:
نشر بصيغ مفتوحة غير مملوكة (CSV, ODS) · توفير رخصة استخدام واضحة · خلو البيانات من أي معلومات شخصية
تاريخ الإصدار: 2023/02 معتمد
STD-STR-06
معيار تخزين وأرشفة البيانات
يحدد المتطلبات الفنية لحفظ البيانات التاريخية وفترات الاستبقاء وآليات الاسترجاع.
المتطلبات الرئيسية:
تحديد وسائط التخزين المناسبة لكل تصنيف · فترات الاحتفاظ حسب المتطلبات النظامية · ضمان سلامة البيانات أثناء الحفظ الطويل
تاريخ الإصدار: 2024/06 قيد المراجعة
